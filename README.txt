Структура папок:
classes - все созданные вами классы
config - здесь все настройки, локализации, доступ к бд и прочее
modules - здесь внешние модули, пока еще не переделанные под наши нужды (не переписанные под классовую систему)
functions - отдельные функции, настолько важные и большие, что требуют отдельной папки и внимания
install - папка для скриптов установки, т.е. создания БД, adminpanel'и  и прочее
components - итоговые странички, которые будут открываться в меню
template - шаблон, css-ки и картинки, которые определяют оформление сайта
admin - панель администрирования и все что связано с ней
